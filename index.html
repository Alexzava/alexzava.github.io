Some of my projects

These are some of my open source projects available on GitHub.

Krypto for Android

Krypto for Android is a simple and secure app to protect your private documents with strong encryption.

It's designed to be simple but secure by using XChaCha20-Poly1305 for symmetric encryption, Argon2id for key derivation function, X25519 (Curve 25519) for secure key exchange.

Windows and Linux versions are coming soon.

Open on GitHub

Italia Covid-19

Italia Covid-19 is a website that tracks and visually presents data and trends related to Covid-19 in Italy.

The data is sourced from the public repository of the Italian Civil Protection (Protezione Civile).

Go to Italia Covid 19

Kitty Chat

KittyChat is a secure end-to-end encrypted messaging application for Android and Web.

It protects your conversations with strong encryption, using the XSalsa20 stream cipher and ECC Curve25519. The server is designed to ensure that no private data is stored, safeguarding users' identities.
