Krypto

Krypto is a simple tool to protect your private documents with encryption.

It's designed to be simple and secure by using AES cipher with a key of 256bit and Argon2ID as key derivation function.

Open on GitHub

Kitty Chat

KittyChat is a secure End-to-End encrypted chat application for Android and Web.

It protect your chats with strong encryption using XSalsa20 stream cipher and ECC Curve25519. The server is designed not to keep track of any private data that could compromise users' identities.

Read the white paper

Italia Covid-19
